# Calculadora

## Descrição

Este projeto consiste em uma aplicação de calculadora desenvolvida para realizar operações matemáticas básicas de forma rápida, simples e eficiente. A aplicação possui interface interativa e é executada em ambiente de desenvolvimento local, sendo ideal para fins educacionais e prática de desenvolvimento web.

## Funcionalidades

- Realizar operações matemáticas básicas:
  - Adição
  - Subtração
  - Multiplicação
  - Divisão
- Interface simples e intuitiva
- Execução em ambiente local via navegador

## Como executar o projeto

Siga os passos abaixo para executar o projeto corretamente:

1. Abra o terminal.
2. Navegue até a pasta onde o projeto está localizado:

   ```bash
   cd nome-da-pasta-do-projeto
   ```

3. Execute o comando abaixo para iniciar o servidor de desenvolvimento:

   ```bash
   npm run dev
   ```

4. Após a execução do comando, pressione a tecla:

   ```bash
   o
   ```

5. O projeto será aberto automaticamente no navegador padrão.
